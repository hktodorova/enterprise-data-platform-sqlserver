# Enterprise Data Platform - SQL Server

This repository is a hands-on project focused on SQL Server, data integration and operational database engineering.

The goal is to build a small but realistic data platform step by step, including:

- SQL Server database design
- Python-based data loading
- monitoring and health checks
- performance tuning examples
- backup and recovery scripts
- Docker-based local setup

The project is still in progress. I am using it to document and practice topics that are relevant for data platform and database engineering roles.

## Planned stack

- Microsoft SQL Server
- Python
- Docker
- T-SQL
- pytest
- PowerShell
